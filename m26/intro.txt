===============================
0. Basic of DOM (Intro Class)
===============================

What is DOM?
DOM হলো document এর জন্য একটি programming interface.

যখন একটি webpage browser এ load হয় তখন HTML document টা DOM tree তে convert হয়।
DOM এর মাধ্যমে JavaScript খুব সহজে HTML elements manipulate করতে পারে।

JavaScript DOM দিয়ে কী কী করতে পারে?
- HTML elements পরিবর্তন করতে পারে
- HTML elements এর CSS style পরিবর্তন করতে পারে
- নতুন HTML element তৈরি করতে পারে
- HTML elements remove করতে পারে
- HTML attribute পরিবর্তন করতে পারে
- নতুন HTML events তৈরি করতে পারে

DOM এর full meaning:
D = Document → file/document
O = Object → elements (html, head, body, title, text)
M = Model → layout structure (tree model)

HTML থেকে DOM Tree কিভাবে হয়?
Document
 └── html
      └── body
           ├── h1
           │    └── "Hello"
           └── p
                └── "World"

Node কী?
Node হলো DOM Tree এর প্রতিটি অংশের নাম।
Document
 └── html
      └── body
           └── ul (Element Node, id="places")
                ├── li (Element Node)
                │    └── "Dhaka" (Text Node)
                └── li (Element Node)
                     └── "Khulna" (Text Node)

ParentNode এবং ChildNode:
- Root node ছাড়া প্রতিটি node এর একটি parent থাকে।
- একটি parent node এর একাধিক child node থাকতে পারে।
- একই parent এর child node গুলোকে sibling বলা হয়।

DOM Methods:
1. getElementById → Id দিয়ে element access করা যায়।
   Example:
   const headingOne = document.getElementById("headingOne");
   headingOne.style.color = "red";

2. getElementsByClassName → Class name দিয়ে একাধিক elements access করা যায়।
   Example:
   const collections = document.getElementsByClassName("paragraph");
   collections[0].style.backgroundColor = "yellow";

3. createElement + appendChild → নতুন element তৈরি করে parent node এ যোগ করা যায়।
   Example:
   const myUl = document.getElementById("my_ul");
   const myNewList = document.createElement("li");
   myNewList.innerText = "I am three";
   myUl.appendChild(myNewList);

===============================
1. Events in Web & JS Event Handling
===============================

Event কী?
- Event হলো কোনো action বা ঘটনা যা ওয়েবপেজে ঘটে।
- উদাহরণ: user interaction (click, input, hover), browser action (load, resize)।
- DOM event system ব্যবহার করে এগুলো detect ও handle করা যায়।

Web এ Event এর ধরন:
1. Mouse Events → click, dblclick, mouseover, mouseout, mousemove
2. Keyboard Events → keydown, keyup, keypress
3. Form Events → submit, change, focus, blur
4. Window/Document Events → load, resize, scroll
5. Clipboard Events → copy, cut, paste
6. Touch Events (Mobile) → touchstart, touchmove, touchend

JavaScript এ Event Handling এর 3টি প্রধান উপায়:
1. Inline Event Handler (HTML attribute)
   Example: <button onclick="alert('Clicked!')">Click Me</button>

2. DOM Property Method
   Example:
   const btn = document.getElementById("myBtn");
   btn.onclick = function() {
     alert("Button clicked!");
   };

3. addEventListener() Method (Best Practice)
   Example:
   const btn = document.getElementById("myBtn");
   btn.addEventListener("click", function() {
     alert("Button clicked via addEventListener!");
   });

Summary:
- Event হলো ওয়েবপেজে ঘটে যাওয়া action বা ঘটনা।
- Event এর ধরন → Mouse, Keyboard, Form, Window, Clipboard, Touch।
- Event Handling এর 3টি উপায় → Inline, Property, addEventListener()।

===============================
2. Add onclick Handler (Directly or via JS)
===============================

Option 1: Inline Event Handler (HTML attribute)
Example:
<button onclick="console.log(7)">Click me</button>
<button onclick="console.log(67)">Another button</button>
<button onclick="console.log('I am clicked')">Third Button</button>
সুবিধা → সহজ ও দ্রুত।
অসুবিধা → HTML ও JS মিশে যায়, maintainability কমে।

Option 2: Function Call via Inline Handler
Example:
function makeYellow() {
  document.body.style.backgroundColor = "yellow";
}
function makeRed() {
  document.body.style.backgroundColor = "red";
}
সুবিধা → Reusable function, multiple জায়গায় ব্যবহার করা যায়।

Option 3: Get Element by ID & Set onclick (Anonymous Function)
Example:
const btnMakeBlue = document.getElementById("btn-make-blue");
btnMakeBlue.onclick = function makeBlue() {
  document.body.style.backgroundColor = "blue";
};
সুবিধা → JS আলাদা থাকে, HTML clean থাকে।

Option 4: Assign Function Reference
Example:
const btnMakePurple = document.getElementById("btn-make-purple");
btnMakePurple.onclick = makePurple;

function makePurple() {
  document.body.style.backgroundColor = "purple";
}
সুবিধা → Function reusable, code পরিষ্কার।

Summary:
- Option 1 → Inline handler (HTML attribute)
- Option 2 → Inline handler + function call
- Option 3 → JS দিয়ে element খুঁজে নিয়ে onclick property সেট করা
- Option 4 → Function reference দিয়ে onclick assign করা
- Inline handler সহজ, কিন্তু maintainability কম। Property method clean ও reusable।

Event Handling in JavaScript (Module Notes)

Cue (Questions):
- HTML element-এ event add করার বিভিন্ন উপায় কী?
- কোনটা best practice এবং কেন?
- Common types of JS events কী কী?

Notes:
1. Inline Event Handler (HTML attribute)
   - সরাসরি HTML element-এর মধ্যে event লেখা হয়।
   - Example: <button onclick="alert('Clicked!')">Click Me</button>
   - অসুবিধা: HTML ও JS মিশে যায়, maintainability কমে।

2. DOM Property Method
   - JavaScript দিয়ে element খুঁজে নিয়ে property (onclick, onchange) সেট করা হয়।
   - Example:
     const btn = document.getElementById('myBtn');
     btn.onclick = function() {
       alert('Button clicked!');
     };
   - অসুবিধা: একবারে শুধু একটি handler assign করা যায় (overwrite হয়)।

3. addEventListener() Method (Best Practice)
   - Modern ও flexible উপায়।
   - Syntax: element.addEventListener("event", callback)
   - Example:
     const btn = document.getElementById('myBtn');
     btn.addEventListener('click', function() {
       alert('Button clicked via addEventListener!');
     });
   - সুবিধা:
     - একাধিক handler attach করা যায়।
     - Event bubbling/capturing control করা যায়।
     - HTML ও JS আলাদা থাকে, clean code।
     - Modern browsers support করে।

Why addEventListener is Best Practice:
- Inline handler → messy, hard to maintain।
- Property method → overwrite problem।
- addEventListener → reusable, scalable, multiple handlers, modern standard।

Common Types of JS Events:
- Mouse Events → click, dblclick, mouseover, mouseout, mousemove
- Keyboard Events → keydown, keyup, keypress
- Form Events → submit, change, focus, blur
- Window/Document Events → load, resize, scroll

Summary:
- Event add করার ৩টা উপায় → Inline handler, DOM property method, addEventListener()
- Best practice → addEventListener()
- Common events → Mouse, Keyboard, Form, Window

Event Handlers & Anonymous Functions

Cue (Questions):
- Event দিয়ে innerText কিভাবে update করা যায়?
- Anonymous Function কী?
- নামহীন / নামবিহীন function এর ব্যবহার কী?

Notes:

1. Event দিয়ে innerText update
   - Event handler ব্যবহার করে HTML element-এর text পরিবর্তন করা যায়।
   - Example:
     <button id="btn-update">Click Me</button>
     <p id="text">Old Text</p>

     const btn = document.getElementById("btn-update");
     const para = document.getElementById("text");

     btn.addEventListener("click", function() {
       para.innerText = "Text updated by Event!";
     });

   - এখানে button এ click করলে paragraph-এর innerText পরিবর্তন হবে।

2. Anonymous Function কী?
   - Anonymous function হলো এমন function যার কোনো নাম থাকে না।
   - সাধারণত event handler বা callback হিসেবে ব্যবহার করা হয়।
   - Example:
     document.getElementById("btn").onclick = function() {
       console.log("Button clicked!");
     };

   - এখানে function-এর কোনো নাম নেই, কিন্তু কাজ করছে।

3. নামহীন / নামবিহীন function এর ব্যবহার
   - Temporary কাজের জন্য ব্যবহার হয়।
   - Event handler, callback, setTimeout/setInterval এ বেশি ব্যবহৃত হয়।
   - Example:
     setTimeout(function() {
       console.log("Executed after 2 seconds");
     }, 2000);

Summary:
- Event handler দিয়ে innerText update করা যায় → element.innerText = "new text"
- Anonymous function হলো নামহীন function, সাধারণত event handler বা callback হিসেবে ব্যবহৃত হয়।
- নামহীন function temporary কাজের জন্য উপযোগী, reusable function এর জন্য named function ভালো।

Explore More Event Types
===============================

Cue (Questions):
- JavaScript এ আর কী কী event type আছে?
- Mouse, Keyboard, Form event কিভাবে কাজ করে?
- Github এর মতো delete confirmation কিভাবে বানানো যায়?

Notes:

1. Mouse Events
   - mouseenter → element এর উপর mouse আসলে trigger হয়।
   - mousemove → element এর উপর mouse move করলে trigger হয়।
   - mouseout → element এর উপর থেকে mouse চলে গেলে trigger হয়।
   Example:
     document.getElementById('btn-mouse')
       .addEventListener('mouseout', function() {
         console.log('event trigger');
       });

2. Focus & Blur Events
   - focus → input box এ cursor গেলে trigger হয়।
   - blur → input box থেকে cursor বের হলে trigger হয়।
   Example:
     document.getElementById('user-name')
       .addEventListener('focus', function() {
         console.log('user about to write name');
       });

     document.getElementById('user-name')
       .addEventListener('blur', function() {
         console.log('user done with name');
       });

3. Keyboard Events
   - keydown → key চাপা মাত্র trigger হয়।
   - keyup → key ছেড়ে দিলে trigger হয়।
   - event.target.value → input box এ user যা লিখছে তা পাওয়া যায়।
   Example:
     document.getElementById('user-name')
       .addEventListener('keyup', function(event) {
         console.log('typing', event.target.value);
       });

===============================
Simple Github-like Delete Confirmation
===============================

Steps:
1. Input box এ user কে "delete" লিখতে হবে।
2. যদি ঠিক "delete" লেখা হয়:
   - Delete button enable হবে।
   - Secret info hide হয়ে যাবে।
3. অন্য কিছু লিখলে:
   - Delete button আবার disable হয়ে যাবে।

Example Code:
document.getElementById('input-delete')
  .addEventListener('keyup', function(event) {
    const text = event.target.value;
    const btnDelete = document.getElementById('btn-delete');

    if(text === 'delete') {
      btnDelete.removeAttribute('disabled');
      const secretTag = document.getElementById('secret-info');
      secretTag.style.display = 'none';
    }
    else {
      btnDelete.setAttribute('disabled', true);
    }
  });

===============================
7. Event Bubble and Stop Propagation
===============================

Cue (Questions):
- Event bubbling কীভাবে কাজ করে?
- stopPropagation() এবং stopImmediatePropagation() এর পার্থক্য কী?
- Event bubble diagram কেমন হয়?

Notes:

1. Event Bubbling কী?
   - যখন কোনো child element এ event ঘটে, তখন সেই event parent → grandparent → root পর্যন্ত propagate হয়।
   - অর্থাৎ event উপরের দিকে bubble করে যায়।
   - Example: li → ol → section → body

2. Event Bubble Diagram (ASCII)
   [body]
     |
     └── [section]
           |
           └── [ol]
                 |
                 └── [li]

   - যদি li তে click হয়:
     Step 1: li handler trigger হবে
     Step 2: ol handler trigger হবে
     Step 3: section handler trigger হবে
     Step 4: body handler trigger হবে

3. stopPropagation()
   - Event কে parent এ propagate হওয়া থেকে থামায়।
   - Example:
     document.getElementById('item-2')
       .addEventListener('click', function(event) {
         console.log('item 2 clicked');
         event.stopPropagation();
       });

   - এখানে li তে click হলে শুধু li handler চলবে, parent এ যাবে না।

4. stopImmediatePropagation()
   - একই element এ একাধিক handler থাকলে প্রথম handler চলার পর অন্য handler গুলোও থেমে যায়।
   - Example:
     document.getElementById('item-2')
       .addEventListener('click', function(event) {
         console.log('item 2 clicked');
         event.stopImmediatePropagation();
       });

   - এখানে li তে click হলে শুধু প্রথম handler চলবে, একই li এর অন্য handler আর চলবে না।

5. Multiple Handlers Example:
   - item-2 এ একাধিক handler আছে।
   - stopImmediatePropagation দিলে → শুধু প্রথম handler চলবে।
   - stopPropagation দিলে → সব handler চলবে কিন্তু parent এ যাবে না।

Summary:
- Event bubbling → child থেকে parent পর্যন্ত event propagate হয়।
- stopPropagation() → parent এ propagate হওয়া বন্ধ করে।
- stopImmediatePropagation() → একই element এর অন্য handler গুলোও বন্ধ করে।
- Diagram: li → ol → section → body

===============================
8. Event Delegate and Benefit of Event Bubble
===============================

Cue (Questions):
- Event delegate কী?
- কেন event bubble উপকারী?
- Event delegate এর সুবিধা কী?
- Beginner-friendly example কেমন হবে?

Notes:

1. Event Delegate কী?
   - Event delegate হলো একটি technique যেখানে child element গুলোর জন্য আলাদা আলাদা event handler না দিয়ে parent element এ একটি handler বসানো হয়।
   - Event bubbling এর কারণে child element এ event ঘটলে parent এ পৌঁছায়।
   - Parent এর handler event.target ব্যবহার করে child কে identify করতে পারে।

2. Benefit of Event Bubble
   - Event bubble এর কারণে parent এ বসানো handler child এর event ধরতে পারে।
   - নতুন dynamically added child element গুলোও একই handler দ্বারা handle হয়।
   - কোড কম হয়, performance ভালো হয়, maintain করা সহজ হয়।

3. Example (Without Delegate)
   - প্রতিটি li এর জন্য আলাদা handler বসাতে হয়।
   - সমস্যা: নতুন li add করলে delete কাজ করে না।

   const items = document.getElementsByClassName('item');
   for(const item of items){
     item.addEventListener('click', function(event){
       event.target.parentNode.removeChild(event.target);
     });
   }

4. Example (With Delegate)
   - শুধু parent ol এ handler বসানো হয়।
   - সব li click করলে delete হবে, নতুন li add হলেও কাজ করবে।

   document.getElementById('item-list')
     .addEventListener('click', function(event){
       event.target.parentNode.removeChild(event.target);
     });

5. Add New Item Example
   - Button click করলে নতুন li তৈরি হয় এবং ol এ append হয়।
   - Delegate থাকার কারণে নতুন item এও click করলে delete হবে।

   document.getElementById('btn-add-item')
     .addEventListener('click', function(){
       const ol = document.getElementById('item-list');
       const li = document.createElement('li');
       li.classList.add('item');
       li.innerText = 'Brand new item added';
       ol.appendChild(li);
     });

Summary:
- Event delegate → parent এ handler বসিয়ে child এর event handle করা।
- Benefit → কম কোড, নতুন element এও কাজ করে, performance ভালো।
- Event bubble এর কারণে delegate কাজ করে।
- Example → ol এ handler বসানো হলে সব li delete করা যায়, নতুন li add হলেও কাজ করে।

26-9 All about Events (Module Summary)


===============================
All About Events (Module Summary)
===============================

Cue (Questions):
- Common HTML events কী কী?
- Event handler কীভাবে কাজ করে?
- addEventListener কীভাবে কাজ করে?
- Event bubbling এবং capturing কী?
- stopPropagation কীভাবে কাজ করে?
- Event delegate কী?

Notes:

1. Common HTML Events
   - onclick → কোনো element এ click করলে trigger হয়।
   - onchange → input বা select এর মান পরিবর্তন হলে trigger হয়।
   - onmouseover → mouse element এর উপর গেলে trigger হয়।
   - onkeydown → keyboard এর key চাপলে trigger হয়।
   - onload → page load হলে trigger হয়।
   - onblur → input box থেকে cursor বের হলে trigger হয়।

2. Event Handler
   - Event handler হলো function যা কোনো event ঘটলে execute হয়।
   - Example:
     <button onclick="addName()">Add Name</button>
     function addName() {
       const nameInput = document.getElementById("name-input");
       const newName = nameInput.value;
       const nameContainer = document.getElementById("name-container");
       const p = document.createElement("p");
       p.innerText = newName;
       nameContainer.appendChild(p);
       nameInput.value = "";
     }

3. addEventListener
   - Modern ও flexible উপায় event handle করার জন্য।
   - Syntax: element.addEventListener("event", callback);
   - Example:
     document.getElementById("btn-add-name")
       .addEventListener("click", function() {
         console.log("Button clicked via addEventListener");
       });

4. Event Bubbling
   - Event child থেকে parent পর্যন্ত propagate হয়।
   - Example:
     li → ul → section → body
   - যদি li তে click হয়, parent গুলোতেও event trigger হয়।

5. Event Capturing
   - Bubbling এর বিপরীত।
   - Event root থেকে child পর্যন্ত propagate হয়।
   - addEventListener এর third parameter true দিলে capturing mode এ কাজ করে।
   - Example:
     element.addEventListener("click", handler, true);

6. Stop Propagation
   - Event কে parent এ propagate হওয়া থেকে থামায়।
   - Example:
     element.addEventListener("click", function(event) {
       event.stopPropagation();
     });

   - stopImmediatePropagation → একই element এর অন্য handler গুলোও বন্ধ করে।

7. Event Delegate
   - Parent এ handler বসিয়ে child এর event handle করা।
   - Benefit → কম কোড, নতুন element এও কাজ করে।
   - Example:
     document.getElementById("item-list")
       .addEventListener("click", function(event) {
         event.target.parentNode.removeChild(event.target);
       });

Summary:
- Common events → onclick, onchange, onmouseover, onkeydown, onload, onblur
- Event handler → function যা event ঘটলে execute হয়
- addEventListener → best practice, multiple handler support করে
- Event bubbling → child থেকে parent এ propagate
- Event capturing → parent থেকে child এ propagate
- stopPropagation → bubbling/capturing থামায়
- Event delegate → parent এ handler বসিয়ে child এর event handle করা যায়

26-10 Basic JS Events all in One

===============================
Basic JavaScript Events (All in One)
===============================

1. What are Events in Web?
- Event হলো কোনো action বা ঘটনা যা ওয়েবপেজে ঘটে।
- উদাহরণ: user interaction (click, input, hover), browser action (load, resize), বা system-generated ঘটনা।
- JavaScript event system ব্যবহার করে এগুলো detect ও handle করা যায়।

2. Different Types of Events
- Mouse Events → click, dblclick, mouseover, mouseout, mousemove
- Keyboard Events → keydown, keyup, keypress
- Form Events → submit, change, focus, blur
- Window/Document Events → load, resize, scroll
- Clipboard Events → copy, cut, paste
- Touch Events → touchstart, touchmove, touchend

3. Signature of addEventListener
element.addEventListener(eventType, callbackFunction, useCapture);
- eventType → string (যেমন "click", "keydown")
- callbackFunction → function যা event ঘটলে execute হবে
- useCapture → optional boolean (default false → bubbling, true → capturing)

4. onClick Handler vs addEventListener
- onClick handler:
  btn.onclick = function() { ... }
  → একবারে শুধু একটি handler assign করা যায়।
- addEventListener:
  btn.addEventListener("click", function() { ... })
  → একাধিক handler attach করা যায়, flexible, modern best practice।

5. Dispatching an Event / Event Chain / Event Propagation
- যখন কোনো event ঘটে, সেটা DOM tree এর মাধ্যমে propagate হয়।
- Propagation এর দুই ধাপ:
  a) Capturing phase → root থেকে child পর্যন্ত যায়।
  b) Bubbling phase → child থেকে root পর্যন্ত যায়।

6. Event Capture vs Event Bubble
- Capture → parent থেকে child এ propagate হয়।
- Bubble → child থেকে parent এ propagate হয়।
- Default behavior → bubbling।

7. Stop Event Propagation
- event.stopPropagation() → parent এ propagate হওয়া বন্ধ করে।
- event.stopImmediatePropagation() → একই element এর অন্য handler গুলোও বন্ধ করে।

8. Event Delegation
- Parent এ handler বসিয়ে child এর event handle করা।
- Benefit → কম কোড, নতুন element এও কাজ করে, performance ভালো।

9. 3 Steps of Event Delegation
Step 1: Parent element এ event listener বসানো।
Step 2: event.target ব্যবহার করে কোন child এ event ঘটেছে তা identify করা।
Step 3: সেই child element এর উপর প্রয়োজনীয় action নেওয়া।

10. JS Event Loop
- Event loop হলো JavaScript এর concurrency model।
- Call stack → synchronous code execute করে।
- Event queue → asynchronous callback গুলো রাখে।
- Event loop → stack খালি হলে queue থেকে callback নিয়ে execute করে।
- এর ফলে JavaScript single-threaded হলেও asynchronous কাজ করতে পারে।

===============================
Summary
===============================
- Events হলো ওয়েবপেজে ঘটে যাওয়া action।
- addEventListener → best practice, multiple handler support করে।
- Event propagation → capturing এবং bubbling।
- stopPropagation → bubbling/capturing থামায়।
- Event delegation → parent এ handler বসিয়ে child এর event handle করা।
- JS Event loop → asynchronous কাজ manage করে।
